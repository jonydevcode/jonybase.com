# jonybase.com

A minimal personal site and blog built with [Astro](https://astro.build/).

## Commands

- `pnpm dev` starts the local development server.
- `pnpm build` checks and builds the site, then generates the production and
  development search indexes.
- `pnpm preview` previews the production build.
- `pnpm deploy` deploys the static build with Wrangler.

Published posts are available as an RSS feed at `/rss.xml`.
